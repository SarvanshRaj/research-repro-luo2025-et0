## Target Environment

- Python 3.10+
- PyTorch ≥2.0 (CPU/GPU)
- TensorFlow ≥2.15 (TFLite export)
- scikit-learn ≥1.3
- numpy ≥1.24
- pandas ≥2.0
- matplotlib ≥3.7
- seaborn ≥0.12
- tqdm ≥4.65
- scipy ≥1.10

## Notes

- All packages available via pip on Colab Free
- TFLite runtime can be installed separately: `pip install tflite-runtime`
- No paid APIs or closed-source dependencies
